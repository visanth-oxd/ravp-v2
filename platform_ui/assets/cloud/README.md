# Cloud platform icons

Place your **GCP**, **AWS**, and **Azure** logo files here. They will be shown in the header under "Cloud native platform".

## Expected files

| File name(s)   | Platform (slot)   | Format        |
|----------------|-------------------|---------------|
| `gcp` or `gke` | Google Cloud      | SVG or PNG    |
| `aws`          | Amazon Web Services | SVG or PNG |
| `azure` or `aks` | Microsoft Azure | SVG or PNG    |

- Use **SVG** for best quality at any size (recommended).
- PNG is also supported (e.g. `gke.png`, `aws.png`, `aks.png`). For each slot the app tries the primary name first, then the alternate (e.g. `gcp` then `gke`).
- Suggested size: at least 64Ã—64 px for PNG; SVG can be any size.

## Where to get logos

- **Google Cloud:** [Google Cloud branding](https://cloud.google.com/use-of-brand-assets)
- **AWS:** [AWS architecture icons / brand](https://aws.amazon.com/architecture/icons/)
- **Azure:** [Microsoft Azure branding](https://www.microsoft.com/en-us/legal/intellectualproperty/trademarks)

Use official brand guidelines when possible. If a file is missing, the app will not show that logo.
